# KitchenIQ

KitchenIQ is an AI-powered recipe recommendation application that helps students and young adults reduce food waste by suggesting recipes based on ingredients they already have at home.

## Features
- Recipe recommendations using available ingredients
- Prioritizes ingredients nearing expiry
- Supports dietary preferences and allergies
- Suggests three ranked recipes
- Simple step-by-step cooking instructions

## Technologies
- AI-powered recipe generation
- Web application
- GitHub for version control
